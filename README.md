# M-Box

M-Box is a Flutter application built with Dart that provides a clean and responsive user experience across multiple platforms.

## Features

* Cross-platform Flutter application
* Clean and responsive UI
* Fast performance
* Easy-to-maintain project structure
* Android, iOS, Web, Windows, Linux, and macOS support

## Tech Stack

* Flutter
* Dart

## Project Structure

```bash
lib/
 └── main.dart        # Main application entry point
assets/               # Application assets
android/              # Android platform files
ios/                  # iOS platform files
web/                  # Web platform files
windows/              # Windows platform files
linux/                # Linux platform files
macos/                # macOS platform files
```

## Getting Started

### Prerequisites

Make sure you have the following installed:

* Flutter SDK
* Dart SDK
* Android Studio or VS Code
* Git

## Installation

Clone the repository:

```bash
https://github.com/ibrahim-codes1/Flutter-mBox.git
```

Go to the project directory:

```bash
cd m_box
```

Install dependencies:

```bash
flutter pub get
```

Run the application:

```bash
flutter run
```

## Build APK

```bash
flutter build apk
```

## Build Web Version

```bash
flutter build web
```
